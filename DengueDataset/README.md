# Dengue Hospitalization and Environmental Time Series Dataset (Brazil)

## General Description

This repository provides a collection of **weekly time series datasets** related to **dengue hospitalization rates** and **socioeconomic, demographic, and environmental variables** for Brazilian municipalities.

The dataset was designed to support **time series forecasting**, **machine learning**, and **deep learning** studies, particularly focusing on **multivariate models** applied to epidemiological data.

The data were used in analyses involving multi-horizon forecasting, model comparison, and spatio-temporal evaluation across municipalities.
